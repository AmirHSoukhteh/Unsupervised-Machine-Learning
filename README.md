### Machine Learning
Machine learning (ML) is a way to train software, called a model, to make predictions or generate content using data. ML systems can be categorized as supervised, unsupervised, reinforcement, or generative AI, each learning differently.

+ The goal of ML is: learn the relationship between input and output of an unknown function form data.
